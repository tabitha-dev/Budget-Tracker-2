# Budget-Tracker-2


Budget Tracker is a simple web application built with Vue.js and Chart.js that allows you to track your expenses, set a budget limit, and view historical spending. You can add new expenses, delete existing ones, and visualize your expenses using bar and pie charts.

## Features

- Add new expenses with a category and amount.
- Delete existing expenses from the list.
- View total expenses, budget limit, remaining budget, and budget utilization percentage.
- Download your expenses data as a CSV file.
- View historical spending over different months using a line chart.

Usage
Enter your budget limit in the input field at the top of the page.

To add a new expense, fill in the "Expense Category" and "Amount" fields in the form, then click "Add Expense."

The expenses table will display all the added expenses along with a "Delete" button to remove individual expenses.

The budget summary will show your total expenses, budget limit, remaining budget, and budget utilization percentage.

The "Download Data as CSV" button allows you to download your expenses data as a CSV file.

The "Show Historical Spending" button enables you to view historical spending over different months using a line chart.
